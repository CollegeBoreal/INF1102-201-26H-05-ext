# 🎯 Objectif

Ce travail consiste à utiliser les expressions régulières (Regex) pour extraire des informations à partir d’un fichier texte.

---

# ⚙️ Outils utilisés

* Bash (`grep -E`)
* Python (`re`)
* PowerShell

---

# 🔍 Éléments analysés

Le script permet d’extraire :

* Emails
* Numéros de téléphone
* Dates
* Adresses IP
* Codes d’erreur HTTP

---

# 🚀 Exécution

### Bash

```bash
./regex_bash.sh
```

### Python

```bash
python3 regex_python.py
```

### PowerShell

```bash
pwsh ./regex_pwsh.ps1
```
