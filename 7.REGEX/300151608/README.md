## 🎯 Objectif

Ce projet permet d’analyser un fichier log Nginx en utilisant des expressions régulières (Regex) avec Python.

Le script extrait automatiquement :
- Le nombre total de requêtes
- Les codes HTTP (200, 404, 500…)
- Les adresses IP
- Les pages les plus visitées
