# 📐 Somme d'une Suite Arithmétique

Un projet Jupyter Notebook implémentant le calcul de la somme des termes d'une suite arithmétique.

---

## 📋 Description

Ce projet propose deux implémentations en Python permettant de calculer la **somme des termes d'une suite arithmétique** à partir de la formule mathématique :

$$S_n = \frac{n}{2} \times (2a + (n-1)d)$$

ou de manière équivalente :

$$S_n = \frac{n}{2} \times (U_1 + U_n)$$

Où :
- `a` → premier terme de la suite
- `d` → raison (différence commune entre deux termes consécutifs)
- `n` → nombre de termes
- `Uₙ` → dernier terme

## 💡 Exemple d'utilisation

**Entrée :**
```
Premier terme (a) = 2
Raison (d)        = 3
Nombre de termes  = 5
```

**Sortie :**
```
U1 = 2.0
U2 = 5.0
U3 = 8.0
U4 = 11.0
U5 = 14.0

Dernier terme      : 14.0
Somme des 5 termes : 40.0
```
